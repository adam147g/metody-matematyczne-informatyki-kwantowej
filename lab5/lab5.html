a) Na czym polega problem Deutcha?¬∂

Problem Deutcha polega na rozstrzygniƒôciu, czy funkcja logiczna f: {0,1} ‚Üí {0,1} jest:

  • sta≈Ça (constant), tzn. f(0) = f(1)
  • zr√≥≈ºnicowana (balanced), tzn. f(0) ‚â† f(1)

W klasycznym podej≈õciu musimy obliczyƒá dwa wyniki: f(0) i f(1), co wymaga dw√≥ch zapyta≈Ñ. W podej≈õciu kwantowym, dziƒôki wykorzystaniu kwantowej superpozycji i interferencji, wystarczy jedno zapytanie do funkcji f (po wykonaniu odpowiednich operacji na kwantowym komputerze), aby rozstrzygnƒÖƒá, czy funkcja jest sta≈Ça, czy zr√≥≈ºnicowana.

b) Gdzie sƒÖ cztery bramki Uf i do jakich grup nale≈ºƒÖ?¬∂

Problem Deutcha dotyczy czterech mo≈ºliwych funkcji f, kt√≥re mogƒÖ byƒá:

Nazwa funkcji f(0) f(1) Typ Operator Uf
f‚ÇÄ(x) = 0 0 0 constant Identity
f‚ÇÅ(x) = 1 1 1 constant CNOT
f‚ÇÇ(x) = x 0 1 balanced X(y) + CNOT
f‚ÇÉ(x) = ¬¨x 1 0 balanced X(y)

Ka≈ºda bramka Uf to operator kwantowy dzia≈ÇajƒÖcy na dw√≥ch qbitach:
|x‚ü©|y‚ü© ‚Üí |x‚ü©|y ‚äï f(x)‚ü©

  • Sta≈Çe funkcje (f‚ÇÄ, f‚ÇÅ) odpowiadajƒÖ operatorom to≈ºsamo≈õciowym lub negujƒÖcym drugi qbit.
  • Funkcje zr√≥≈ºnicowane (f‚ÇÇ, f‚ÇÉ) odpowiadajƒÖ operacjom CNOT (kontrola na pierwszym qbiczie).

def classical_solution(secret_number):
    num_bits = secret_number.bit_length()
    query = 0
    for i in range(num_bits):
        if (secret_number >> i) & 1:
            query |= (1 << i)
    return query

def bernstein_vazirani(s):
    """Algorytm Bernsteina-Vaziraniego."""
    n = len(s)
    qc = QuantumCircuit(n + 1, n)

    qc.x(n)
    qc.barrier()
    for qubit in range(n+1):
        qc.h(qubit)
    qc.barrier()

    oracle(qc, s, n)
    qc.barrier()

    for qubit in range(n+1):
        qc.h(qubit)

    qc.barrier()

    qc.measure(range(n), range(n))

    return qc
